# VERIFICATION TEMPLATES

Post-completion checklists to validate work quality. Use after the goal workflow completes.

## Available Templates

| Template | Use After | Focus |
|----------|-----------|-------|
| [feature-verification](feature-verification.md) | `feature-request` goal | Acceptance criteria, integration, UX |
| [bug-verification](bug-verification.md) | `bug-investigation` goal | Root cause confirmed, regression prevented |
| [security-verification](security-verification.md) | Security-related work | Threat model, attack surface, data protection |
| [performance-verification](performance-verification.md) | Performance optimization | Metrics, benchmarks, no regressions |

## How to Use

Attach the relevant template after completing your goal workflow:

```
Backend Engineer + Bug Investigation + Deep Investigator + Bug Verification
→ Investigate root cause, fix, then verify with structured checklist
```
