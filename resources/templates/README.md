# Templates

Markdown templates for the `forge-context/` directory. Used by `goals/project-init.md` to bootstrap project state.

## Available Templates

| Template | Purpose |
|----------|---------|
| [PROJECT.md](PROJECT.md) | Project identity, constraints, principles |
| [STATE.md](STATE.md) | Current progress, blockers, next actions |
| [ROADMAP.md](ROADMAP.md) | Phased plan with deliverables and success criteria |

## Usage

These templates are **starting points**, not rigid schemas. Adapt sections to fit your project:

- Remove sections that don't apply
- Add project-specific sections as needed
- Keep each file under 200 lines — concise beats comprehensive
